## 🚀 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on Amazon Prime Movies & TV Shows dataset to uncover patterns, trends, and insights related to content distribution, ratings, and audience engagement.

---

## 🎯 Business Objective

* Analyze distribution of Movies vs TV Shows
* Identify high-performing genres
* Understand content growth trends
* Study relationship between ratings, runtime & popularity
* Generate insights to improve content strategy

---

## 📂 Dataset

The dataset is created by merging:

* `titles.csv` → Content details
* `credits.csv` → Cast & crew details

### 🔑 Key Features:

* 🎬 Title, Type, Description
* 📅 Release Year
* ⭐ IMDb & TMDb Ratings
* ⏱ Runtime
* 🌍 Production Countries
* 👥 Cast & Crew

---

## 🧹 Data Cleaning & Preparation

✔ Merged datasets using `id`
✔ Handled missing values
✔ Filled `seasons` with 0
✔ Replaced categorical nulls with *Unknown*
✔ Removed duplicates
✔ Ensured dataset is **analysis-ready & error-free**

## 📊 Exploratory Data Analysis

### 🔹 Univariate Analysis

* Content Type Distribution
* Release Year Distribution
* IMDb Score Distribution
* Runtime Analysis
* Role Distribution

### 🔹 Bivariate Analysis

* Type vs IMDb Score
* Runtime vs Ratings
* Content Growth Over Time
* Type vs Age Certification
* IMDb Votes Distribution
* Production Countries

### 🔹 Multivariate Analysis

* Average Ratings by Content Type
* Actor Frequency Analysis
* Feature Relationships (Heatmap)

### 🔹 Correlation Analysis

* Numerical feature relationships using correlation matrix

## 📈 Key Insights

📌 Movies dominate the platform
📌 Content production increased after 2015
📌 Ratings mostly fall in mid-to-high range
📌 Certain genres perform consistently better
📌 Audience engagement varies by content type
